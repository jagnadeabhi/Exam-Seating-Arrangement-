Exam Seating Arrangement

Exam Hall Seating Arrangement Systems is developed for the college to simplify examination hall allotment and seating arrangemnt.



HOW TO USE? (QUICK GUIDE)

Admin Login

1.Login with clicking Admin Login.
2.Log In with registered username and password.
3.Update Student Information.
4.Data Aloocation.
5.View Particular Hall Arrangement.

A) Update Data
In Admin Section there is authorization to change Hall arrangement, Seat number & Desk number.

B) Data Allocation
you can set or allocate data for student like seat number,hall number & desk number.

C) View
You can view seating arrangement of particular hall.
